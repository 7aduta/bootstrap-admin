bootstrap-admin

A clean, minimalistic WordPress Administration theme inspired from Twitter's Bootstrap http://twitter.github.com/bootstrap

This project was built for the http://magazi.org network of stores. This plugin will continue to be improved as for now just the basic stuff have been changed. If you like it and use it on your projects, consider donating on http://aristeides.com

== Installation ==

Just copy to your plugins folder and activate (or Network activate) it.
